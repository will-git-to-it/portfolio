Portfolio Importer

Imports your portfolio pages as Figma frames (one per page × viewport).
Run the export script first to generate the image files.

1 In your terminal: cd figma-export && npm install && npm run export
2 Select all files inside figma-export/output/ below.
3 Click Import to Figma — frames will appear on the current page.
